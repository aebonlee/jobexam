## 변경 파일 목록

### 1. `src/config/site.js`
- NAV_ITEMS 배열에 과목별학습하기, 학습요약(children 제거), 문제풀기 추가
- 기존 학습요약의 과목 드롭다운 → 과목별학습하기로 이동
- 문제풀기: 필기 CBT (/pilgi), 실기 연습 (/silgi) children 추가

### 2. `src/components/layout/Navbar.jsx`
- **active 상태 로직 수정**: children이 있는 메뉴는 children 경로 매칭, 없는 메뉴는 exact match
  - `/learn/counseling` → 과목별학습하기만 active
  - `/learn` → 학습요약만 active
- **key prop 수정**: `item.path` → `item.label` (과목별학습하기와 학습요약이 동일 path /learn 사용)
- **모바일 메뉴**: mobileSubnavOpen key도 `item.label`로 변경, "전체 과목 보기" 링크 제거

### 3. `src/pages/practice/PracticeHome.jsx` (신규)
- 문제풀기 허브 페이지
- 필기 CBT 카드 → /pilgi (5과목 100문항 4지선다)
- 실기 연습 카드 → /silgi (서술형/계산형)
- learn-card 패턴과 동일한 practice-card 스타일 사용

### 4. `src/App.jsx`
- PracticeHome import 추가
- `/practice` 라우트 추가

### 5. `src/styles/learn.css`
- `.practice-grid`, `.practice-card`, `.practice-card-*` 스타일 추가
- learn-card 패턴 기반으로 일관된 디자인

### 6. `src/styles/responsive.css`
- `.practice-grid` 모바일 반응형 (1fr) 추가

## Active 상태 로직 상세

```jsx
// 기존 (문제: /learn이 접두사라 과목별학습하기와 학습요약 모두 active)
location.pathname.startsWith(item.path) ? 'active' : ''

// 변경 후
const isActive = item.children
  ? item.children.some(child =>
      location.pathname === child.path ||
      location.pathname.startsWith(child.path + '/'))
  : location.pathname === item.path;
```
